## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| n8n -> FastAPI API | n8n sends HTTP requests to FastAPI endpoints on internal Docker network; untrusted if network is exposed |
| FastAPI -> SQLite | ORM queries with user-supplied data from n8n payloads |
| Docker host -> n8n UI | n8n web editor exposed on port 5678; must not be public without auth |
